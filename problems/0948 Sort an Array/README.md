```
Given an array of integers `nums`, sort the array in ascending order.



**Example 1:**

    
    
    **Input:**  [5,2,3,1]

**Example 2:**

**Note:**

  1. `1 <= A.length <= 10000`
  2. `-50000 <= A[i] <= 50000`
```
